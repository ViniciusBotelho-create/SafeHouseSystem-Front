# SafeHouse Web

Front-end of the SafeHouse System, a residential expense control application.
Built with React, TypeScript and Vite, consuming the [SafeHouse API](https://github.com/ViniciusBotelho-create/SafeHouseSystem.git).

## Technologies

- React 18
- TypeScript
- Vite
- React Router DOM
- Axios
- Tailwind CSS

## Prerequisites

- Node.js 18+
- SafeHouse API running locally
